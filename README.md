# Glimmer Handoff Package

**Glimmer** — The beautiful, public-API-powered, literature-rich, doom-free scroll for your free moments.

Pure positive micro-delights, wonder, humor, cosmic views, and public domain literature & poetry — no algorithm, no drain, just lighter.

This repo contains the complete, production-ready handoff for building Glimmer as a Next.js (or heroic single-file) web app / PWA. Optimized for vibe coders and AI agents (Claude Projects/Artifacts, Grok Build, Cursor, Codex-style workflows).

## Quick Start for Codex / AI Builder
1. Clone or use this repo as context.
2. Start with the Master Context Prompt (in /prompts/).
3. Follow phases sequentially, pasting each prompt into your AI coding environment.
4. After each phase, test and return here with output/errors for refinements.

## What Makes Glimmer Special
- Jam-packed with reliable public + public-domain APIs (jokes, quotes, cute animals, facts, NASA cosmic, weather contextual, **Project Gutenberg literature/poetry/short stories**, Open Library, Standard Ebooks quality texts, Advice Slip, BoredAPI sparks, and more).
- Rich, varied, beautiful vertical feed that feels alive and never repetitive.
- Optional contextual weather/location theming.
- **Glimmer Reader**: Calm, typography-first modal reader for public domain excerpts and full short works (custom beautiful reader + optional EPUB.js wiring for high-quality editions).
- Ethical, user-controlled "doom-free" design: Load more on demand, favorites, share, gentle exit prompts.
- Instant one-button entry + immersive scroll mode.
- Local-first (prefs, favorites, recent history in localStorage; offline cache possible).
- Sovereign & delightful — aligns with HarrowHaus values.
